# 烘焙基础笔记

{{< callout type="info" >}}
目标：把烘焙当成“可复现的实验”——称量、温度、时间、记录。
{{< /callout >}}

## 1. 常见原料与作用
- 面粉：
  - 低筋：更松软（蛋糕/饼干）
  - 中筋：通用（面包也可）
  - 高筋：更筋道（面包）
- 糖：甜度 + 上色 + 保湿
- 脂肪（黄油/油）：香气 + 酥松
- 鸡蛋：结构 + 蓬松 + 乳化
- 泡打粉/小苏打：膨松（注意与酸性材料匹配）

## 2. 关键变量
- 温度：室温黄油更易打发；面糊温度影响起发
- 含水量：稀/稠 → 组织不同
- 搅拌：
  - 过度搅拌会出筋（变韧）
  - 翻拌要轻，避免消泡

## 3. 失败排查
- 蛋糕塌陷：未熟透/消泡/温度骤降
- 饼干摊大：黄油太软、糖比例高、烤箱偏低
- 面包不发：酵母失活、温度不对、盐接触酵母

# 披萨面团与烘烤

{{< callout type="info" >}}
目标：底部脆、边缘蓬松；把“温度”当作关键调味。
{{< /callout >}}

## 1. 面团关键参数
- 含水量：60-70%（越高越难操作但更蓬松）
- 盐：2-2.5%（筋性+风味）
- 酵母：少量长发酵更香
- 冷发：冰箱 24-72h 风味更好

## 2. 基础流程
1) 混合：面粉 + 水 + 酵母（可静置 20min 自溶）
2) 加盐揉/折叠至光滑
3) 一发：室温至 1.5-2 倍
4) 分割滚圆：冷藏发酵
5) 回温：使用前 1-2h
6) 开饼：从中心往外推；尽量不擀掉边缘气泡

## 3. 烘烤要点
- 烤箱预热：最高温 30min+（最好有披萨石/钢）
- 上料：酱少量，芝士不要堆太厚（否则出水）
- 时间：高温短烤；观察边缘上色

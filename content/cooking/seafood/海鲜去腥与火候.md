# 海鲜去腥与火候

{{< callout type="info" >}}
目标：保留鲜甜，减少腥味；火候宁短勿老。
{{< /callout >}}

## 1. 去腥的几个“杠杆”
- 新鲜度：越新鲜越少处理
- 盐：提前轻盐腌（5-15min）帮助脱水、提味
- 酒/姜/葱：挥发性香气掩盖腥
- 酸：柠檬/醋在出锅前或蘸料用（避免久煮发涩）

## 2. 常见食材火候
- 虾：变红卷曲即可；过久会粉
- 鱼片：轻轻变白即熟；可用“余温”带熟
- 贝类：开口即熟，没开口的丢掉
- 鱿鱼：要么短（30-60s），要么长（炖煮），中间段最韧

## 3. 简单做法模板
### 3.1 清蒸鱼
- 抹盐 + 姜葱垫底
- 大火上汽后蒸：每 500g 约 6-8 分钟（按厚度调整）
- 出锅淋热油 + 生抽

### 3.2 黄油蒜香虾
- 虾擦干 → 大火快煎 → 黄油蒜末下锅 30s → 撒盐/黑胡椒

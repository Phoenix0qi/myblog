# 跑步训练记录

{{< callout type="info" >}}
目标：稳定跑量、降低受伤风险、用最小计划换取可持续。
{{< /callout >}}

## 1. 基础指标（每次记录）
- 日期 / 天气 / 跑鞋
- 距离（km）/ 时长 / 配速（min/km）
- 心率：平均 / 最大（如有）
- 主观强度：RPE（1-10）
- 感受：呼吸、腿部、足底、膝/髋

## 2. 训练类型模板
### 2.1 轻松跑（E）
- 目的：打底、恢复
- 强度：能完整说话（RPE 3-4）
- 要点：步频自然、落地轻、全程放松

### 2.2 节奏跑（T）
- 目的：乳酸阈值
- 强度：能说短句（RPE 6-7）
- 结构：热身 10-15min → T 15-30min → 放松 10min

### 2.3 间歇（I）
- 目的：速度与心肺
- 结构示例：
  - 6×400m（组间慢跑 200m）
  - 4×1km（组间慢跑 2-3min）
- 要点：配速稳定，不要前快后崩

### 2.4 长距离（LSD）
- 目的：耐力与燃脂能力
- 强度：轻松可聊（RPE 4-5）
- 要点：补给/补水提前准备

## 3. 伤病与调整
- 疼痛等级：
  - 0-2：可继续观察
  - 3-5：减少 30-50% 跑量，避免强度
  - 6+：停止跑步，补充力量/就医
- 常见信号：胫骨内侧、髂胫束、足底筋膜

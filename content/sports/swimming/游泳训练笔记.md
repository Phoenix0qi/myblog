# 游泳训练笔记

{{< callout type="info" >}}
目标：技术优先，先“省力”，再“变快”。
{{< /callout >}}

## 1. 技术要点（自由泳）
### 1.1 身体线条
- 头：看下方 45°，避免抬头
- 髋：保持高位，减少“坐水”
- 核心：轻收紧，保持身体像一根杆

### 1.2 划水（Catch/拉水）
- 入水：指尖先入，避免拍水
- 前伸：延伸到位再开始拉
- 抓水：前臂竖直（EVF）尽量早
- 出水：手从大腿旁自然推出

### 1.3 打腿
- 频率：小幅高频，比大幅更省力
- 触发：从髋带动，不是膝盖踢

### 1.4 呼吸
- 吐气：在水下连续吐
- 侧头：一只眼出水即可
- 节奏：2/3/4 拍呼吸按训练目标选择

## 2. 训练结构模板
- 热身：200-400m（混合泳姿/划臂/打腿）
- 技术：4×50m（每组专注 1 个要点）
- 主项：
  - 有氧：10×100m（间歇 15-25s）
  - 阈值：6×100m（间歇 10-15s）
  - 速度：12×25m（全力，间歇 30-45s）
- 放松：100-200m

## 3. 常用训练（Drills）
- 单臂划水：感受身体滚转
- 6-1-6：6 次打腿 + 1 次换臂
- 指尖拖水：保持高肘
- 抱板打腿：注意不塌腰

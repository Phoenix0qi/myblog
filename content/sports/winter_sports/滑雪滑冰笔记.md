# 滑雪/滑冰笔记

{{< callout type="warning" >}}
安全优先：护具、热身、量力而行；不适立即停止。
{{< /callout >}}

## 1. 装备清单
- 保暖：分层（排汗层/保暖层/防风防水层）
- 手套/护具：护膝、护腕（新手建议）
- 眼镜：防雾、防紫外
- 其他：暖宝宝、补水、能量棒

## 2. 滑雪（单板/双板通用基础）
### 2.1 姿势
- 膝踝微屈，髋在脚上方
- 重心：保持在板/雪鞋中部

### 2.2 控速与转弯（核心）
- 控速靠“走刃/刹车”，不是硬扛
- 先练：横滑 → J 形弯 → 连续 S 弯
- 关键：
  - 视线看向下一个方向
  - 身体跟随板的方向，避免上半身扭着

### 2.3 常见问题
- 后仰：会加速且难转弯 → 贴靴舌、髋前移
- 害怕：速度上来动作变形 → 放慢、缩短坡度、分段练

## 3. 滑冰（基础）
- 站姿：膝盖盖住脚尖，身体前倾
- 推蹬：外侧刃发力，脚尖不外撇太夸张
- 刹车：先学犁式/外刃刹，再练 T 刹

#####  1.4.2. <a name='-1'></a>练习感悟

{{% details title="练习感悟" %}}

  {{% details title="解锁指板"  closed="true" %}}
  - ⚠️ 解锁指板
    - 每天把 **每个音/音程** 照顾一般
    - 开放世界**地图解锁**；CAGED 表示区域分布关系；
  - ⚠️ 音名 与 音级； 记忆与练习
    - 关系与理解思路
      - 坐标系/地图 思路：
        - 音名 == World coordinate system == 绝对位置
        - 音级 scale/mode == Local coordinate system == 相对关系
        - 每一个调都是新的局部坐标系，坐标系转换，旋转矩阵？；
      - 模式匹配 思路
        - 任意给定 音级 和 音的位置 note coordinate(TAB coordinate)，快速模式匹配 Pattern Recognization, which CAGED pattern, which scale position, 
        - CAGED coordinate system; 
    - 优点
      - 记忆指板音名，快速反应音级，转调更快；
      - 指板不记音名 只记相对关系，容易迷失方向位置；
      - 更容易找到和识别和弦扩展音；
      - **大局观 —— 管中窥豹**
      - 结论：**首调(相对关系) + 固定调(绝对位置)** 都要记； mode 以主音音程记忆为主，每个mode都是全新的音阶 
    - 练习
      - 随机出调内音名，**音名反应音级**、**和弦CAGED五个位置按法**;
        - 看音名，快速反应音 是 调的第几级音（练习固定调，音程部分练习）：**音名反应音级**
      - 和弦位置练习
  - ⚠️ 其他
    - 音程 不同音起始，像强调该音为主音的调，像调式音阶的感觉；
    - 夯实基础后，大量曲目练习

  - ⚠️ 自然大调 自然小调 (主音音程) 两个极端 向其他scale/mode收敛渐进练习

  {{% details title="音程练习2-2注意事项"  closed="true" %}}
  - ⚠️ 思考几件事
    - **首调音程**：
      - 音阶不同音级起始的自然音程
        - 任意音起始的音程按法，**绝对发射**
      - 帮助记忆不同级数和弦 扩展音是什么，
      - 例： 2m9: 9=1; 5dom13: 13=3 ……
    - **音名音级对应**
      - 不同调的音级音名是什么
      - 例： G: G A B C D E #F
    - 思考音所在的**整体位置**
      - CAGED位置
      - 音阶指形位置
      - 例：C大调，D=2 在5弦上的CAGED， 1M - A shape； 音阶指形 La 指形
  - ⚠️ 练习顺序
    - 练习1： 
      - **遍历CAGED位置优先**；练习位置转换；
      - 12-13... × 5 
      - 1-base CAGED 五个位置后，练习2-base, 3-base ……
    - 练习2： 
      - **遍历1-2-3……-base优先**，再练习其他位置; 练习同位置内音阶记忆；
      - 12-13... 22-23 ... 同一个位置，再练 5 个 CAGED位置
  {{% /details %}}

  {{% details title="CAGED 3-2 注意事项"  closed="true" %}}

    - 1）根音音阶连接 
    - 2）根音+五度 + 音阶连接 到下一个和弦
  - ⚠️ **地点检索，坐标索引**
    - 给定音名/音级，迅速反应所在box(CAGED)和弦按法，音阶位置(音阶指形)
  {{% /details %}}

  {{% details title="《GG-1》五声音阶 注意事项"  closed="true" %}}

    - 2) 推到下个音，连接回上一个box的音
  - ⚠️ box之间连接; position-shifting
    - 1) GG-1 position-shifting 4.22 拓展
      - 全把位 相邻两根弦；横向地图薄弱
